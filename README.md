# Codepred Announcements API

---

## Requirements

- Java 17  
- Jakarta Validation  
- H2 (in-memory database)  
- JPA (Hibernate)  
- Gradle  
- Postman  

---

## Running the Application

1. Clone the repository:

   ```bash
   git clone https://github.com/kamiloses/Codepred.git
   ```
   
 
2. Open the file:
   ```bash
   cd codepred
   ```

3. Run all tests (optional):

   ```bash
   gradlew test
   
 4. Start the application:

     ```bash
      gradlew bootRun
      ```
